# Repositorio de Prácticas JavaScript - Master en Frontend Asistido con IA

Este repositorio forma parte del programa de DEV.F Fundación Traxión Tech Challenge 2025. Ha sido seleccionado para recibir la beca de Extensión, cubriendo el 100% del costo del programa para completar 6 módulos adicionales y obtener el Master en Frontend Asistido con IA.

Este repositorio tiene como finalidad ser un espacio de aprendizaje y práctica de JavaScript, organizando los ejercicios y proyectos desarrollados durante el programa.


## Estructura del Repositorio

El contenido principal de las prácticas se encuentra en el directorio `mis-practicas-intro_js/`. Dentro de este, las prácticas están organizadas cronológicamente en subdirectorios que siguen el formato `X-js-DD-MM-AAAA`, donde:
- `X` es un número secuencial para la práctica.
- `DD-MM-AAAA` es la fecha en que se realizó o realizara la práctica.

Cada uno de estos subdirectorios contiene los archivos (`.html`, `.js`, etc.) correspondientes a los ejercicios de ese día. Adicionalmente, si una lección incluye un ejercicio práctico o un entregable, este se encontrará dentro de una subcarpeta llamada **`entregable`** dentro del directorio de la lección correspondiente.

### Arquitectura Visual
```
2-Web-B-Master-JavaScript/
├── .gitignore
├── README.md
└── mis-practicas-intro_js/
    └── [numero]-js-[DD-MM-AAAA]/  // Carpeta de la lección
        ├── [archivo-de-practica].js
        ├── index.html
        └── entregable/              // Carpeta para el entregable
            └── [archivo-entregable].js
```

### Ejemplos
- `mis-practicas-intro_js/1-js-27-01-2025/`: Prácticas de JavaScript del 27 de enero de 2025.
- `mis-practicas-intro_js/2-js-29-01-2025/`: Prácticas de JavaScript del 29 de enero de 2025.
- `mis-practicas-intro_js/3-js-03-02-2026/`: Prácticas de JavaScript del 03 de febrero de 2026.
- ...y así sucesivamente para las demás fechas.
